# Amazon Sales Dashboard (India)

## Project Overview
This project presents an interactive Amazon India Sales Dashboard developed using Tableau. The dashboard provides insights into sales trends, product performance, shipping status, and regional sales distribution.

## Tools Used
- Tableau Desktop
- Microsoft Excel / CSV

## Dataset
Amazon India Sales Dataset

## Dashboard Features
- KPI Cards
- Weekly Sales Trend
- Category-wise Quantity Analysis
- Order Status Analysis
- State-wise Sales Map
- Sales by Size
- Sales Channel Distribution
- Interactive Filters

## Key Insights
- Sales performance across weeks
- Top performing states
- Order fulfilment status
- Product category analysis
- Size-wise sales distribution
